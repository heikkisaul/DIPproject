Object tracking with location approximation

Heikki Saul, Taavi Adamson
Advisor: Gholamreza Anbarjafari, PhD

This application requires Python 3.x (assured compatibility with 3.5.2), OpenCV 3.x (assured compatibility with 3.1.0)
and numpy 1.x (assured compatibility with 1.11.1).

All files in /src must be placed into one local catalog for the program to work.
Calibration for HSV must be done manually, by editing HSV_LOWER and HSV_UPPER variables in main.py.
When using multiple cameras, the correct one must be selected by editing CAMERA_ID value in main.py

To start the application, please run main.py.
